# home-design-server
基于SpringBoot+Vue前后端分离的宅心艺家装网系统（源码+文档），系统包含管理员、设计师、用户角色，功能如下： 管理员：首页统计（案例统计、预约统计）；用户管理；设计名师；风格管理；设计案例；预约管理；评价管理；商品种类；商品管理；订单管理；评论管理；平台留言；公告信息；个人信息；密码修改。 设计师：设计案例；预约管理；评价管理；公告信息；个人信息（上传头像、证书）；修改密码； 用户：注册、登录系统；首页；设计案例（详情、收藏）；设计名师（详情、预约、评价）；家装商城（详情、加入购物车、下单、评论）；平台留言；系统通知。用户还具有个人中心功能，功能包括：案例收藏、名师预约、名师评价、我的订单、商品收藏、商品评论、平台留言、个人资料、修改密码。
